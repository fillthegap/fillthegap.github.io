Robocup Asia-Pacific @ Dubai

In December 1st we were told by our CS teacher that there will be a Robocup held in Dubai, I was really intrested and since the material they introduced was mostly graphical with some Arduino code in it, I decided to join although I had limited experience in both of these subjects.

The school created a workshop + election process at friday, which consisted of using the code to run virtual robots in an area, which was powered by Microsoft Robotics suit and Nvidia PhysX. My team was elected to participate, since we have done the fastest lap by combining ultrasonic sensivity and follow the line sensors. (It was quite easy since we used a graphical user interface to program the virtual robots, which then translated into C++ code that the virtual bot executed.

You can see some useful info in Event Website and Event video.
We won the election process and went for the master competition held in Dubai Knowledge village.

The beginning of the event was easy since it was quite similar to what our school trained us for but the final levels were quite hard as the code should both run on the Physical and Virtual robot which was quite scary as sometimes the one that worked on the virtual bot didn’t work on physical and vice versa.


After lots of challenge our team Pioneers got the first place in U19 challenge which was quite nice, although we asked why the code gave different results on Physical and Virual, the company supplier told us that in order to make the virtual section more challenging they assigned some random values to initial velocity which gave weird results everytime in return.
